# Nexus Vital

Landing Page desenvolvida como protótipo para apresentação do Método Nexus Vital.

## Objetivo

Criar uma página moderna e responsiva para apresentar o método, gerar autoridade e direcionar potenciais clientes para o atendimento via WhatsApp.

- CSS3
- JavaScript
- Design Responsivo
